## How to run the dApp

first clone the repo
```shell
$ git clone "https://github.com/mohitchandel/buy-me-coffee-web3.git"
```
Go to the project directory
```bash
$ cd buy-me-coffee-web3
```
Install dependencies
```bash
$ npm i or yarn
```
Run Tests
```bash
$ npx hardhat test
```
Start the server
```bash
$ npm run start
```
